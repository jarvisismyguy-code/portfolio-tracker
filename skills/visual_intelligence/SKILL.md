# Visual Intelligence Skill

## Description
Generates portfolio visualizations from Trading212 holdings data. Creates Sankey diagrams (flow of capital) and Sector Heatmaps. Outputs PNG images for Discord attachment.

## Trigger Conditions
- Daily at 9 AM (alongside daily report)
- On-demand via command

## Inputs
- `portfolio_split.json`: Generated by core tracker with holdings, values, percentages

## Outputs
- `outputs/portfolio_sankey.png`: Sankey diagram showing capital flow
- `outputs/sector_heatmap.png`: Heatmap by sector
- `outputs/holdings_pie.png`: Pie chart by ticker

## Dependencies
- plotly (for Sankey and heatmap)
- pandas (data manipulation)
- kaleido (for static image export)

## Usage
```python
from skills.visual_intelligence.main import generate_all_charts
generate_all_charts()
```

## Image Specs
- Format: PNG
- Resolution: 1200x800
- Discord optimized (compressed)
